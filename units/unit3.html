1. مقدمة إلى JavaScript

شرح المفهوم

JavaScript هي لغة برمجة تستخدم لجعل صفحات الويب تفاعلية. عند استخدام HTML فقط، تكون الصفحة ثابتة، أي أن المحتوى لا يتغير أثناء استخدام الطالب للموقع. أما عند استخدام JavaScript فيمكن للصفحة أن تتفاعل مع المستخدم، وتقرأ البيانات، وتنفذ العمليات الحسابية، وتتخذ قرارات، وتغير محتوى الصفحة مباشرة.

معلومة مهمة تعمل JavaScript غالبًا داخل متصفح المستخدم، ولذلك تسمى برمجة جهة العميل Client-Side.

أهمية JavaScript في صفحات الويب

  • قراءة البيانات من المستخدم.
  • التحقق من صحة الإدخال.
  • تنفيذ العمليات الحسابية.
  • تغيير محتوى الصفحة.
  • إظهار رسائل وتنبيهات.

2. إضافة JavaScript إلى صفحة HTML

شرح المفهوم

لكتابة كود JavaScript داخل صفحة HTML نستخدم الوسم <script>. هذا الوسم يخبر المتصفح أن الكود الموجود داخله هو كود JavaScript.

الهدف من المثال

توضيح كيفية كتابة JavaScript داخل صفحة HTML وعرض نص داخل الصفحة.

المثال

<html> <body> <h2>JavaScript Example</h2> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = "My First JavaScript"; </script> </body> </html>

شرح السطور

السطر 3: عنوان داخل الصفحة.

السطر 4: فقرة HTML تحمل المعرف demo.

السطر 6: بداية كود JavaScript.

السطر 7: الوصول إلى العنصر الذي يحمل المعرف demo وتغيير محتواه باستخدام innerHTML.

المدخل لا يوجد.
المخرج يظهر النص داخل الصفحة: My First JavaScript

أماكن وضع JavaScript

يمكن وضع JavaScript داخل:

  • body: عندما نريد تنفيذ الكود أثناء عرض الصفحة.
  • head: عندما نريد تعريف دوال أو كود سيتم استدعاؤه لاحقًا.

3. الإدخال والإخراج في JavaScript

أولاً: الإدخال باستخدام prompt()

تستخدم الدالة prompt() لطلب إدخال قيمة من المستخدم. عند تنفيذها يظهر مربع حوار يحتوي رسالة وصندوق إدخال.

الهدف من المثال

قراءة اسم المستخدم ثم عرض رسالة ترحيب.

المثال

var name = prompt("Please enter your name:"); alert("Hello " + name);

شرح السطور

السطر 1: يطلب من المستخدم إدخال اسمه ويخزن القيمة في المتغير name.

السطر 2: يعرض رسالة ترحيب تحتوي الاسم الذي أدخله المستخدم.

المدخل اسم المستخدم، مثال: Ahmad
المخرج Hello Ahmad

ثانياً: الإخراج باستخدام alert()

تستخدم الدالة alert() لعرض رسالة للمستخدم في مربع حوار.

alert("Welcome to JavaScript");

الشرح: يظهر مربع حوار يحتوي الرسالة النصية.

المدخل لا يوجد.
المخرج مربع حوار يحتوي: Welcome to JavaScript

ثالثاً: الإخراج باستخدام document.write()

تستخدم الدالة document.write() لكتابة النص داخل الصفحة.

document.write("This is JavaScript Output");

الشرح: يكتب النص داخل صفحة الويب مباشرة.

تنبيه يفضل استخدام document.write في الأمثلة التعليمية البسيطة فقط.

رابعاً: تغيير محتوى عنصر باستخدام innerHTML

<p id="demo"></p> <script> document.getElementById("demo").innerHTML = "Hello JavaScript"; </script>

الشرح: يصل البرنامج إلى العنصر الذي يحمل id=demo ثم يغير النص الموجود بداخله.

4. المتغيرات وأنواع البيانات

شرح المفهوم

المتغير هو مكان في الذاكرة يستخدم لتخزين قيمة يمكن استخدامها لاحقًا داخل البرنامج. في JavaScript يتم تعريف المتغير باستخدام الكلمة var.

الهدف من المثال

تعريف متغيرين وحساب مجموعهما.

المثال

var price1 = 5; var price2 = 6; var total = price1 + price2; alert(total);

السطر 1: تعريف متغير اسمه price1 وتخزين القيمة 5.

السطر 2: تعريف متغير اسمه price2 وتخزين القيمة 6.

السطر 3: إنشاء متغير total لتخزين نتيجة الجمع.

السطر 5: عرض قيمة المجموع.

المدخل لا يوجد.
المخرج 11

  • Number: رقم، مثل: 25

5. تحويل النص إلى رقم باستخدام Number()

السطر 1: قراءة الرقم الأول وتحويله من نص إلى رقم.

السطر 2: قراءة الرقم الثاني وتحويله من نص إلى رقم.

السطر 4: جمع الرقمين.

السطر 6: عرض النتيجة.

6. العمليات الحسابية

شرح المفهوم

تستخدم العمليات الحسابية لإجراء الحسابات داخل البرنامج. من أهم العمليات: الجمع، الطرح، الضرب، القسمة، وباقي القسمة.

السطر 1 و 2: تعريف المتغيرين a و b.

السطر 4: تنفيذ عملية الجمع.

السطر 5: تنفيذ عملية الطرح.

8. الجملة الشرطية if

شرح المفهوم

تستخدم جملة if لتنفيذ أوامر معينة إذا تحقق شرط محدد. إذا كان الشرط صحيحًا ينفذ الكود، وإذا لم يتحقق يتم تجاهله.

الهدف من المثال

قراءة درجة طالب، وإذا كانت أكبر أو تساوي 50 يظهر أنه ناجح.

المثال

var mark = Number(prompt("Enter student mark")); if(mark >= 50){ alert("Pass"); }

شرح السطور

السطر 1: قراءة الدرجة وتحويلها إلى رقم.

السطر 3: فحص إذا كانت الدرجة أكبر أو تساوي 50.

السطر 4: إذا تحقق الشرط يتم عرض Pass.

المدخل 70
المخرج Pass

9. الجملة الشرطية if...else

شرح المفهوم

تستخدم عندما نريد تنفيذ مسارين مختلفين: مسار إذا تحقق الشرط، ومسار آخر إذا لم يتحقق.

الهدف من المثال

تحديد إذا كان الطالب ناجحًا أو راسبًا.

المثال

var mark = Number(prompt("Enter student mark")); if(mark >= 50){ alert("Pass"); } else{ alert("Fail"); }

شرح السطور

if: تفحص الشرط.

else: تنفذ إذا لم يتحقق الشرط السابق.

المدخل 40
المخرج Fail

10. الجملة الشرطية else if

شرح المفهوم

تستخدم else if عندما يكون لدينا أكثر من احتمال أو أكثر من مسار.

الهدف من المثال

تحديد تقدير الطالب حسب درجته.

المثال

var mark = Number(prompt("Enter student mark")); if(mark >= 90){ alert("Excellent"); } else if(mark >= 80){ alert("Very Good"); } else if(mark >= 70){ alert("Good"); } else if(mark >= 50){ alert("Pass"); } else{ alert("Fail"); }

شرح السطور

if: الشرط الأول.

else if: شروط إضافية يتم فحصها بالتتابع.

else: إذا لم يتحقق أي شرط سابق.

المدخل 85
المخرج Very Good

11. معاملات المقارنة

شرح المفهوم

تستخدم معاملات المقارنة داخل الشروط لمعرفة العلاقة بين القيم.

  • == يساوي
  • != لا يساوي
  • > أكبر من
  • < أصغر من
  • >= أكبر أو يساوي
  • <= أصغر أو يساوي

الهدف من المثال

مقارنة رقمين وتحديد أيهما أكبر.

المثال

var num1 = Number(prompt("Enter first number")); var num2 = Number(prompt("Enter second number")); if(num1 > num2){ alert("First number is larger"); } else{ alert("Second number is larger"); }
المدخل 10 و 5
المخرج First number is larger

12. الجملة switch

شرح المفهوم

تستخدم switch عندما نريد اختيار تنفيذ معين بناءً على قيمة متغير واحد مع وجود عدة حالات. وهي بديل مرتب في بعض الحالات عن if المتعددة.

الهدف من المثال

إدخال رقم يوم وإظهار اسم اليوم.

المثال

var day = Number(prompt("Enter day number (1-3)")); switch(day){ case 1: alert("Sunday"); break; case 2: alert("Monday"); break; case 3: alert("Tuesday"); break; default: alert("Invalid day"); }

شرح السطور

switch(day): فحص قيمة المتغير day.

case: حالة معينة.

break: الخروج من switch بعد تنفيذ الحالة.

default: تنفذ إذا لم تتحقق أي حالة.

المدخل 2
المخرج Monday

13. مفهوم الحلقات التكرارية

شرح المفهوم

نستخدم الحلقات لتكرار تنفيذ الأوامر عدة مرات بدلًا من كتابة نفس الكود مرات كثيرة. من أشهر الحلقات في JavaScript:

  • while
  • do...while
  • for

14. حلقة while

شرح المفهوم

تنفذ حلقة while الأوامر طالما أن الشرط صحيح.

الهدف من المثال

طباعة الأرقام من 1 إلى 5.

المثال

var i = 1; while(i <= 5){ alert(i); i++; }

شرح السطور

السطر 1: تعريف عداد يبدأ من 1.

السطر 3: تكرار طالما i أقل أو يساوي 5.

السطر 4: عرض قيمة العداد.

السطر 5: زيادة العداد بمقدار 1.

المدخل لا يوجد.
المخرج 1 ثم 2 ثم 3 ثم 4 ثم 5

15. حلقة do...while

شرح المفهوم

تشبه while لكن الفرق أنها تنفذ الأوامر مرة واحدة على الأقل قبل فحص الشرط.

الهدف من المثال

طلب إدخال رقم أكبر من 10، وإعادة الطلب حتى يتم إدخال رقم صحيح.

المثال

var number; do{ number = Number(prompt("Enter number greater than 10")); } while(number <= 10); alert("Correct number");

شرح السطور

السطر 1: تعريف متغير لتخزين الرقم.

do: بداية تنفيذ الحلقة.

while(number <= 10): إذا كان الرقم أقل أو يساوي 10 تتكرر الحلقة.

المدخل 5 ثم 7 ثم 12
المخرج Correct number

16. حلقة for

شرح المفهوم

تستخدم for عندما يكون عدد مرات التكرار معروفًا مسبقًا. وتتكون من: بداية، شرط، وتحديث.

الهدف من المثال

حساب مجموع الأرقام من 1 إلى 5.

المثال

var sum = 0; for(var i = 1; i <= 5; i++){ sum = sum + i; } alert(sum);

شرح السطور

sum = 0: متغير لتخزين المجموع.

i = 1: بداية العداد.

i <= 5: شرط استمرار الحلقة.

i++: زيادة العداد بعد كل دورة.

sum = sum + i: إضافة قيمة العداد إلى المجموع.

المدخل لا يوجد.
المخرج 15

17. المصفوفات Arrays

شرح المفهوم

المصفوفة تسمح بتخزين عدة قيم داخل متغير واحد. كل عنصر داخل المصفوفة له رقم يسمى الفهرس، ويبدأ العد من الصفر.

إنشاء مصفوفة تحتوي أسماء ثلاث سيارات وعرض أول عنصر.

var cars = ["BMW","Toyota","Volvo"]; alert(cars[0]);

السطر 1: تعريف مصفوفة باسم cars تحتوي ثلاثة عناصر.

السطر 3: عرض العنصر الأول، لأن cars[0] تمثل أول عنصر.

18. استخدام المصفوفات مع الحلقات

شرح المفهوم

غالبًا ما تستخدم المصفوفات مع الحلقات لعرض جميع العناصر بشكل منظم.

الهدف من المثال

عرض جميع عناصر مصفوفة باستخدام حلقة for.

المثال

var cars = ["BMW","Toyota","Volvo"]; for(var i = 0; i < cars.length; i++){ alert(cars[i]); }

شرح السطور

cars.length: عدد عناصر المصفوفة.

cars[i]: الوصول إلى العنصر الحالي حسب قيمة i.

19. الدوال Functions

شرح المفهوم

الدالة هي مجموعة أوامر تنفذ مهمة محددة. نستخدمها لتنظيم الكود وتقليل التكرار وإعادة استخدام الأوامر أكثر من مرة.

الهدف من المثال

إنشاء دالة لحساب حاصل ضرب رقمين.

المثال

function multiply(a,b){ return a * b; } var result = multiply(4,3); alert(result);

شرح السطور

function multiply(a,b): تعريف دالة اسمها multiply تستقبل رقمين.

return a * b: تعيد حاصل ضرب الرقمين.

var result = multiply(4,3): استدعاء الدالة وإرسال القيم 4 و3 إليها.

alert(result): عرض الناتج.

المدخل 4 و 3

السطر 4: تكرار حسب عدد القيم التي أدخلها المستخدم.

السطر 5: قراءة قيمة جديدة في كل دورة.

السطر 6: إضافة القيمة إلى المجموع.

السطر 9: عرض النتيجة النهائية.

22. التمارين مع الحلول

التمرين 1: قراءة اسم المستخدم

اكتب برنامجًا يقرأ اسم المستخدم ويعرض رسالة ترحيب.

var name = prompt("Enter your name"); alert("Welcome " + name);
التمرين 2: جمع رقمين

اكتب برنامجًا يقرأ رقمين ويعرض مجموعهما.

var num1 = Number(prompt("Enter first number")); var num2 = Number(prompt("Enter second number")); var sum = num1 + num2; alert("Sum = " + sum);
التمرين 3: حساب مساحة مستطيل

اقرأ الطول والعرض ثم احسب مساحة المستطيل.

var length = Number(prompt("Enter length")); var width = Number(prompt("Enter width")); var area = length * width; alert("Area = " + area);
التمرين 4: عدد زوجي أو فردي

اكتب برنامجًا يحدد إذا كان العدد زوجيًا أو فرديًا.

var num = Number(prompt("Enter a number")); if(num % 2 == 0){ alert("Even number"); } else{ alert("Odd number"); }
التمرين 5: تحديد نجاح الطالب

إذا كانت الدرجة أكبر أو تساوي 50 يظهر Pass، وإلا يظهر Fail.

var mark = Number(prompt("Enter student mark")); if(mark >= 50){ alert("Pass"); } else{ alert("Fail"); }
التمرين 6: مجموع الأرقام من 1 إلى 10

استخدم حلقة for لحساب مجموع الأرقام من 1 إلى 10.

var sum = 0; for(var i = 1; i <= 10; i++){ sum = sum + i; } alert("Sum = " + sum);
التمرين 7: طباعة الأرقام من 1 إلى 5

استخدم حلقة while لطباعة الأرقام من 1 إلى 5.

var i = 1; while(i <= 5){ document.write(i + "<br>"); i++; }
التمرين 8: حساب متوسط 3 علامات

اقرأ ثلاث علامات ثم احسب المتوسط.

var m1 = Number(prompt("Enter mark 1")); var m2 = Number(prompt("Enter mark 2")); var m3 = Number(prompt("Enter mark 3")); var avg = (m1 + m2 + m3) / 3; alert("Average = " + avg);
التمرين 9: استخدام المصفوفة

أنشئ مصفوفة تحتوي أسماء ثلاث مدن واعرضها باستخدام for.

var cities = ["Hebron","Jerusalem","Ramallah"]; for(var i = 0; i < cities.length; i++){ document.write(cities[i] + "<br>"); }
التمرين 10: دالة لحساب مربع عدد
