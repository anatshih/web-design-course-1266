1. مقدمة إلى JavaScript

JavaScript هي لغة برمجة تستخدم لجعل صفحات الويب تفاعلية. عند استخدام HTML فقط تكون الصفحة ثابتة، أما مع JavaScript فيمكن التفاعل مع المستخدم، وقراءة البيانات، وتنفيذ العمليات، وتغيير المحتوى مباشرة.

💡 معلومة مهمة
تعمل JavaScript داخل متصفح المستخدم، ولذلك تسمى برمجة جهة العميل Client-Side.

أهمية JavaScript

  • قراءة البيانات من المستخدم
  • التحقق من صحة الإدخال
  • تنفيذ العمليات الحسابية
  • تغيير محتوى الصفحة ديناميكياً
  • إظهار رسائل وتنبيهات
🔗

2. إضافة JavaScript إلى صفحة HTML

نستخدم الوسم <script> لكتابة JavaScript داخل HTML. هذا الوسم يخبر المتصفح أن الكود داخله هو JavaScript.

المثال

<html> <body> <h2>JavaScript Example</h2> <p id="demo"></p> <script> document.getElementById("demo").innerHTML = "My First JavaScript"; </script> </body> </html>

شرح السطور

السطر 4

عنوان داخل الصفحة.

السطر 5

فقرة HTML تحمل المعرف demo.

السطر 8

الوصول للعنصر بالمعرف demo وتغيير محتواه عبر innerHTML.

📥 المدخل
لا يوجد.
📤 المخرج
يظهر النص: My First JavaScript
🖥️

3. الإدخال والإخراج في JavaScript

أولاً: الإدخال — prompt()

تعرض مربع حوار يطلب إدخالاً من المستخدم وتعيد القيمة المدخلة.

var name = prompt("Please enter your name:"); alert("Hello " + name);
📥 المدخل
Ahmad
📤 المخرج
Hello Ahmad

ثانياً: الإخراج — alert()

تعرض رسالة للمستخدم داخل مربع حوار.

alert("Welcome to JavaScript");

ثالثاً: الكتابة في الصفحة — document.write()

document.write("This is JavaScript Output");
⚠️ تنبيه
يُفضَّل استخدام document.write في الأمثلة التعليمية البسيطة فقط.

رابعاً: تغيير محتوى عنصر — innerHTML

<p id="demo"></p> <script> document.getElementById("demo").innerHTML = "Hello JavaScript"; </script>
⚡ المقارنة بين طرق الإخراج الثلاث مقارنة مهمة
alert() مربع حوار منبثق
يعرض رسالة في نافذة منبثقة منفصلة
يوقف تنفيذ البرنامج حتى يضغط المستخدم موافق
لا يغير محتوى الصفحة
يظهر نص فقط، بدون تنسيق HTML
✅ مثالي لـ: التنبيهات، تأكيد نتائج، الاختبار السريع
document.write() كتابة مباشرة في الصفحة
يكتب النص مباشرة داخل صفحة الويب
إذا استُدعي بعد تحميل الصفحة يمسح المحتوى كله
لا يمكن تحديد مكان محدد في الصفحة
يدعم كتابة وسوم HTML داخله
⚠️ مثالي لـ: الأمثلة التعليمية البسيطة فقط
innerHTML تعديل عنصر محدد
يغير محتوى عنصر HTML بعينه عبر id
لا يؤثر على بقية محتوى الصفحة
يدعم وسوم HTML داخله
الأسلوب الاحترافي المُفضَّل في المشاريع
🏆 مثالي لـ: المشاريع الحقيقية وتحديث أجزاء الصفحة

أمثلة مقارنة — نفس النتيجة بثلاث طرق

الهدف: عرض الجملة "مرحباً بك في JavaScript" بالطرق الثلاث:

🔔 alert()
alert("مرحباً بك في JavaScript");
📤 النتيجة: نافذة منبثقة تحتوي الرسالة — تتوقف الصفحة حتى يغلقها المستخدم
📄 document.write()
document.write("مرحباً بك في JavaScript");
📤 النتيجة: يُكتب النص داخل الصفحة في مكان تنفيذ السطر
✏️ innerHTML
<p id="out"></p> <script> document.getElementById("out") .innerHTML = "مرحباً بك في JavaScript"; </script>
📤 النتيجة: يُحدَّث العنصر الذي id="out" فقط دون المساس بباقي الصفحة
💡 متى تستخدم كل طريقة؟
  • استخدم alert() عند الحاجة لتنبيه المستخدم أو اختبار قيمة متغير بسرعة.
  • استخدم document.write() في الأمثلة والشرح التعليمي البسيط فقط.
  • استخدم innerHTML في المشاريع الحقيقية لتحديث أجزاء محددة من الصفحة.

5. تحويل النص إلى رقم — Number()

دالة prompt() تعيد القيمة دائماً كنص. لإجراء عمليات حسابية يجب تحويله باستخدام Number().

المثال

var num1 = Number(prompt("Enter first number")); var num2 = Number(prompt("Enter second number")); var sum = num1 + num2; alert(sum);
📥 المدخل
5 و 7
📤 المخرج
12

6. العمليات الحسابية

تستخدم للحسابات الرياضية داخل البرنامج.

+الجمع
-الطرح
*الضرب
/القسمة
%باقي القسمة

المثال

var a = 10; var b = 5; var sum = a + b; // 15 var sub = a - b; // 5 var mul = a * b; // 50 var div = a / b; // 2 var mod = 10 % 3; // 1

7. ترتيب تنفيذ العمليات الحسابية

إذا احتوى التعبير على أكثر من عملية، تنفذ JavaScript حسب ترتيب محدد:

  • الأقواس أولاً
  • الضرب والقسمة
  • الجمع والطرح

المثال

var result = 5 + 3 * 2; alert(result);

يُنفَّذ الضرب أولاً: 3 × 2 = 6، ثم الجمع: 5 + 6 = 11

📤 النتيجة: 11
🔀

8. الجملة الشرطية if

تنفذ أوامر معينة إذا تحقق شرط محدد. إذا لم يتحقق الشرط يتم تجاهل الكود.

المثال — تحديد النجاح

var mark = Number(prompt("Enter student mark")); if(mark >= 50){ alert("Pass"); }
📥 المدخل
70
📤 المخرج
Pass
⚖️

9. الجملة الشرطية if...else

تستخدم عند وجود مسارين: مسار إذا تحقق الشرط، وآخر إذا لم يتحقق.

المثال

var mark = Number(prompt("Enter student mark")); if(mark >= 50){ alert("Pass"); } else{ alert("Fail"); }
📥 المدخل
40
📤 المخرج
Fail
🎯

10. الجملة الشرطية else if

تستخدم عند وجود أكثر من احتمال أو مسار.

المثال — تقدير الطالب

var mark = Number(prompt("Enter student mark")); if(mark >= 90){ alert("Excellent"); } else if(mark >= 80){ alert("Very Good"); } else if(mark >= 70){ alert("Good"); } else if(mark >= 50){ alert("Pass"); } else{ alert("Fail"); }
📥 المدخل
85
📤 المخرج
Very Good
🔍

11. معاملات المقارنة

تستخدم داخل الشروط لمعرفة العلاقة بين القيم.

==يساوي
!=لا يساوي
>أكبر من
<أصغر من
>=أكبر أو يساوي
<=أصغر أو يساوي

المثال

var num1 = Number(prompt("Enter first number")); var num2 = Number(prompt("Enter second number")); if(num1 > num2){ alert("First number is larger"); } else{ alert("Second number is larger"); }

12. الجملة switch

تستخدم عند اختيار تنفيذ معين بناءً على قيمة متغير واحد مع عدة حالات. بديل منظم عن if المتعددة.

المثال — أيام الأسبوع

var day = Number(prompt("Enter day number (1-3)")); switch(day){ case 1: alert("Sunday"); break; case 2: alert("Monday"); break; case 3: alert("Tuesday"); break; default: alert("Invalid day"); }

شرح العناصر

switch(day)

فحص قيمة المتغير day.

case

حالة معينة يتم فحصها.

break

الخروج من switch بعد تنفيذ الحالة.

default

تنفذ إذا لم تتحقق أي حالة.

📥 المدخل
2
📤 المخرج
Monday
🔁

13. مفهوم الحلقات التكرارية

نستخدم الحلقات لتكرار الأوامر عدة مرات بدلاً من كتابة نفس الكود مرات كثيرة.

whileطالما الشرط صحيح
do...whileمرة على الأقل

14. حلقة while

تنفذ الأوامر طالما أن الشرط صحيح.

المثال — طباعة 1 إلى 5

var i = 1; while(i <= 5){ alert(i); i++; }

شرح السطور

i = 1

تعريف عداد يبدأ من 1.

i <= 5

تكرار طالما i أقل أو يساوي 5.

i++

زيادة العداد بمقدار 1 في كل دورة.

📤 المخرج: 1 ثم 2 ثم 3 ثم 4 ثم 5
🔃

15. حلقة do...while

تشبه while لكنها تنفذ الأوامر مرة واحدة على الأقل قبل فحص الشرط.

المثال

var number; do{ number = Number(prompt("Enter number greater than 10")); } while(number <= 10); alert("Correct number");
📥 المدخل
5 ثم 7 ثم 12
📤 المخرج
Correct number
🔢

16. حلقة for

تستخدم عندما يكون عدد مرات التكرار معروفاً مسبقاً. تتكون من: بداية، شرط، تحديث.

المثال — مجموع 1 إلى 5

var sum = 0; for(var i = 1; i <= 5; i++){ sum = sum + i; } alert(sum);

شرح السطور

i = 1

بداية العداد.

i <= 5

شرط استمرار الحلقة.

i++

زيادة العداد بعد كل دورة.

sum = sum + i

إضافة قيمة العداد للمجموع.

📤 المخرج: 15
📋

17. المصفوفات Arrays

تسمح بتخزين عدة قيم داخل متغير واحد. كل عنصر له رقم فهرس يبدأ من الصفر.

المثال

var cars = ["BMW", "Toyota", "Volvo"]; alert(cars[0]);

شرح السطور

cars[0]

العنصر الأول (BMW) — الفهرس يبدأ من 0.

cars[1]

العنصر الثاني (Toyota).

cars[2]

العنصر الثالث (Volvo).

📤 المخرج: BMW
🔄

18. استخدام المصفوفات مع الحلقات

غالباً تستخدم المصفوفات مع الحلقات لعرض جميع العناصر بشكل منظم.

المثال

var cars = ["BMW", "Toyota", "Volvo"]; for(var i = 0; i < cars.length; i++){ alert(cars[i]); }

شرح السطور

cars.length

عدد عناصر المصفوفة (3).

cars[i]

الوصول للعنصر الحالي حسب قيمة i.

📤 المخرج: BMW ثم Toyota ثم Volvo

19. الدوال Functions

الدالة مجموعة أوامر تنفذ مهمة محددة. تستخدم لتنظيم الكود وتقليل التكرار وإعادة الاستخدام.

المثال — دالة الضرب

function multiply(a, b){ return a * b; } var result = multiply(4, 3); alert(result);

شرح السطور

function multiply(a,b)

تعريف دالة اسمها multiply تستقبل رقمين.

return a * b

تعيد حاصل ضرب الرقمين.

multiply(4, 3)

استدعاء الدالة بالقيمتين 4 و3.

📤 المخرج: 12

22. التمارين مع الحلول

انقر على أي تمرين لعرض الحل.

التمرين 1 — قراءة اسم المستخدم

اكتب برنامجاً يقرأ اسم المستخدم ويعرض رسالة ترحيب.

var name = prompt("Enter your name"); alert("Welcome " + name);
التمرين 2 — جمع رقمين

اكتب برنامجاً يقرأ رقمين ويعرض مجموعهما.

var num1 = Number(prompt("Enter first number")); var num2 = Number(prompt("Enter second number")); var sum = num1 + num2; alert("Sum = " + sum);
التمرين 3 — حساب مساحة مستطيل

اقرأ الطول والعرض ثم احسب مساحة المستطيل.

var length = Number(prompt("Enter length")); var width = Number(prompt("Enter width")); var area = length * width; alert("Area = " + area);
التمرين 4 — عدد زوجي أو فردي

اكتب برنامجاً يحدد إذا كان العدد زوجياً أو فردياً.

var num = Number(prompt("Enter a number")); if(num % 2 == 0){ alert("Even number"); } else{ alert("Odd number"); }
التمرين 5 — تحديد نجاح الطالب

إذا كانت الدرجة ≥ 50 يظهر Pass، وإلا Fail.

var mark = Number(prompt("Enter student mark")); if(mark >= 50){ alert("Pass"); } else{ alert("Fail"); }
التمرين 6 — مجموع 1 إلى 10

استخدم حلقة for لحساب مجموع الأرقام من 1 إلى 10.

var sum = 0; for(var i = 1; i <= 10; i++){ sum = sum + i; } alert("Sum = " + sum);
التمرين 7 — طباعة 1 إلى 5 بـ while

استخدم حلقة while لطباعة الأرقام من 1 إلى 5.

var i = 1; while(i <= 5){ document.write(i + "<br>"); i++; }
التمرين 8 — حساب متوسط 3 علامات

اقرأ ثلاث علامات ثم احسب المتوسط.

var m1 = Number(prompt("Enter mark 1")); var m2 = Number(prompt("Enter mark 2")); var m3 = Number(prompt("Enter mark 3")); var avg = (m1 + m2 + m3) / 3; alert("Average = " + avg);
التمرين 9 — استخدام المصفوفة

أنشئ مصفوفة تحتوي أسماء ثلاث مدن واعرضها باستخدام for.

var cities = ["Hebron", "Jerusalem", "Ramallah"]; for(var i = 0; i < cities.length; i++){ document.write(cities[i] + "<br>"); }
التمرين 10 — دالة لحساب مربع عدد

اكتب دالة تستقبل رقماً وتعيد مربعه.

function square(num){ return num * num; } var result = square(5); alert(result);
