🧩 أولاً: واجهة المستخدم الرسومية GUI

واجهة المستخدم الرسومية هي العناصر التي يراها المستخدم ويتفاعل معها داخل الصفحة، مثل الأزرار وحقول النصوص ومناطق النصوص وصناديق الاختيار وقوائم الاختيار.

Button Text Textarea Checkbox Select

ما المقصود بـ GUI؟

هي الواجهة التي تسمح للمستخدم بإدخال البيانات أو اختيار أمر أو مشاهدة نتيجة. كل عنصر من عناصر الواجهة يمكن التعامل معه برمجياً باستخدام JavaScript.

لماذا نستخدمها؟

  • لتسهيل التفاعل مع صفحة الويب.
  • لإدخال البيانات والحصول على نتائج.
  • لربط العناصر بأحداث مثل النقر أو التغيير.

أهم فكرة يجب فهمها

العناصر ليست مجرد شكل في الصفحة، بل هي كائنات لها خصائص مثل value وchecked وselectedIndex، ولها أحداث مثل onclick وonchange.

عندما ينقر المستخدم على زر، تستطيع JavaScript تنفيذ دالة محددة. وعندما يكتب في حقل نص أو يختار من قائمة، نستطيع قراءة ذلك التغيير.

✅ ثانياً: التحقق من صحة البيانات Form Validation

قبل إرسال البيانات إلى الخادم، يجب التأكد من أنها موجودة وصحيحة ومنطقية. فمثلاً لا يجوز أن يكون العمر فارغًا أو نصًا غير رقمي أو عددًا غير منطقي.

length isNaN() حقول إجبارية مدى القيم

لماذا نتحقق من البيانات؟

  • لمنع إرسال بيانات ناقصة أو خاطئة.
  • لتقليل العبء على الخادم.
  • لتحسين تجربة المستخدم بإخباره بالخطأ فورًا.

ماذا نفحص غالبًا؟

  • هل الحقل فارغ؟
  • هل القيمة رقمية أو نصية حسب المطلوب؟
  • هل القيمة ضمن مدى منطقي؟

⚡ ثالثاً: صفحات الويب الديناميكية DHTML

DHTML ليس لغة مستقلة، بل هو مصطلح يشير إلى مجموعة تقنيات تجعل الصفحة أكثر ديناميكية، مثل HTML وCSS وJavaScript وDOM.

HTML CSS JavaScript DOM

الفكرة الأساسية

الصفحة الديناميكية لا تبقى ثابتة كما هي، بل تتغير عند التفاعل معها: قد يتغير نص، أو لون، أو حجم صورة، أو موضع عنصر، دون الحاجة إلى فتح صفحة جديدة.

ما الفرق بين التغيير والاستبدال؟

أحيانًا نستبدل الصفحة كلها، وهذا ليس الأفضل غالبًا. الأفضل هو تعديل جزء محدد داخل الصفحة نفسها، مثل تغيير عنوان أو إضافة رسالة أو تكبير صورة.

➡ السابق: Validation التالي: DOM ⬅

🧠 رابعاً: نموذج كائن الوثيقة DOM

DOM يسمح لنا بالتعامل مع عناصر الصفحة على أنها كائنات لها خصائص وإجراءات، ويمكن الوصول إليها غالبًا باستخدام document.getElementById().

getElementById() innerHTML src width / height style

ما الذي يقدمه DOM؟

  • الوصول لأي عنصر في الصفحة بالاسم المعرف id.
  • تغيير النصوص والصور والأحجام والألوان والمواقع.
  • إنشاء صفحات أكثر ديناميكية دون إعادة تحميلها.

مثال بسيط جدًا

إذا كان لدينا عنصر يحمل id="title"، يمكننا تغيير محتواه هكذا:

document.getElementById("title").innerHTML = "عنوان جديد";

🖱️ خامساً: التعامل مع الأحداث Events

الحدث هو شيء يحدث داخل الصفحة، مثل تحميل الصفحة، أو النقر بالفأرة، أو مرور المؤشر فوق عنصر. عند وقوع الحدث، يمكن تنفيذ دالة برمجية.

onload onclick onmouseover onmouseout onmousedown onmouseup

أحداث الصفحات

onload يحدث عند انتهاء تحميل الصفحة. أما onunload فيحدث عند مغادرتها. تستخدم هذه الأحداث لتشغيل رسالة ترحيب أو بدء حركة أو تهيئة بعض العناصر.

أحداث الفأرة

أحداث الفأرة مهمة جدًا في بناء التفاعل. مثال: onclick عند النقر، وonmouseover عند مرور المؤشر، وonmouseout عند ابتعاده.

➡ السابق: DOM التالي: Animation ⬅

🎈 سادساً: الكائنات المتحركة Animation / Moving Objects

بعد فهمنا للأحداث وDOM، نستطيع تحريك العناصر داخل الصفحة بتغيير مواقعها أو أحجامها مع التحكم بسرعة التنفيذ باستخدام دوال مثل setTimeout وclearTimeout.

setTimeout() clearTimeout() style.left style.top width / height

كيف تحدث الحركة؟

الحركة لا تعني أن العنصر يتحرك وحده، بل إننا نغيّر موضعه أو حجمه مرات متتابعة بسرعة مناسبة، فيبدو للمستخدم وكأنه يتحرك.

ما الأداتان الأهم؟

  • setTimeout: لاستدعاء دالة بعد مدة زمنية قصيرة.
  • clearTimeout: لإيقاف الحركة أو التوقيت.

📝 المراجعة النهائية للوحدة

إذا فهمت النقاط التالية، فأنت على الطريق الصحيح لفهم الوحدة الرابعة:

  • تعرفت على عناصر الواجهة GUI وكيف يتفاعل المستخدم معها.
  • فهمت كيف نتحقق من صحة البيانات قبل الإرسال.
  • أدركت أن DHTML هو استخدام مجموعة تقنيات لجعل الصفحة ديناميكية.
  • تعلمت أن DOM يمكّننا من الوصول لعناصر الصفحة وتغييرها.
  • فهمت دور الأحداث مثل onclick وonmouseover وonload.
  • رأيت كيف يمكن تحريك العناصر باستخدام الزمن وتغيير المواقع.
اختبار قصير نهائي

1) أي مفهوم يتيح لنا تغيير نص عنصر داخل الصفحة؟

2) أي حدث يناسب تنفيذ أمر عند النقر على زر؟

3) أي دالة مناسبة لفحص أن القيمة ليست رقمًا؟

4) أي دالة تستخدم غالبًا لإيقاف الحركة؟

أجب عن الأسئلة ثم اضغط على "احسب نتيجتي".
