السؤال 1: معرفة الخطأ (Debugging)

الهدف: اكتشاف أخطاء HTML/CSS الزمن المقترح: 25 دقيقة

المطلوب من الطالب في السؤال 1:

  • اقرأ الكود المعطى في المثال.
  • حدّد عدد الأخطاء (حسب عنوان المثال) ومكان كل خطأ.
  • اكتب التصحيح المقترح لكل خطأ (شفهيًا أو على ورقة/ملف مؤقت).
  • اختياري: نفّذ التصحيح على جهازك وشغّل الصفحة للتأكد.
قاعدة: لا تغيّر فكرة المثال، فقط أصلح الأخطاء. ركّز على الوسوم، الإغلاق، الخصائص (Attributes)، والروابط.

السؤال 2: معرفة المخرجات (Predict the Output)

الهدف: توقع ما يظهر وما السلوك الزمن المقترح: 20 دقيقة

المطلوب من الطالب في السؤال 2:

  • اقرأ الكود كاملًا.
  • اكتب/اذكر: ماذا سيظهر على الشاشة؟ وكيف سيكون الشكل؟
  • إذا يوجد “سلوك” (روابط/نموذج): ماذا يحدث عند الضغط أو الإرسال؟
تنبيه: لا تشغّل الكود مباشرة في البداية. توقّع أولًا ثم شغّل للتحقق.

السؤال 3: كتابة برمجية (مشروع مصغّر داخل المختبر)

الهدف: بناء صفحة كاملة الزمن المقترح: 35–40 دقيقة

المطلوب من الطالب في السؤال 3:

  • إنشاء ملفين: index.html وstyle.css.
  • كتابة صفحة واحدة تجمع: نصوص + قائمة + روابط + جدول + نموذج + CSS.
  • لا يوجد تسليم — سيتم عرض العمل للمدرس داخل المختبر.
قاعدة التقييم داخل المختبر: كل نقطة من قائمة الفحص = إنجاز واضح على الشاشة.

خطوات التنفيذ المطلوبة

1) أنشئ ملفين داخل مجلد باسمك:

  • index.html
  • style.css

2) في index.html ضع الأقسام التالية (بالترتيب):

  1. عنوان رئيسي: h1 باسم المقرر.
  2. فقرة تعريف قصيرة (سطرين).
  3. صورة img مع alt و width="220".
  4. قائمة ul فيها 4 عناصر (مهارات/مواضيع).
  5. جدول (3 أعمدة × 3 صفوف بيانات) بعنوان “خطة الأسبوع”.
  6. رابط داخلي في أعلى الصفحة يقفز إلى قسم النموذج (مثلاً href="#form").
  7. نموذج form يحتوي:
    • الاسم (required)
    • البريد (type="email" + required)
    • تخصص (select 3 خيارات)
    • رسالة (textarea + required)
    • زر إرسال + زر مسح

3) في style.css طبّق تنسيق خفيف:

  • خط مناسب + padding عام.
  • بطاقة/حاوية div class="card" بخلفية فاتحة وحدود.
  • تنسيق الجدول (حدود + تباعد + تمييز صف العنوان).
  • تنسيق النموذج (عرض الحقول 100% + مسافات).
  • RTL عبر direction: rtl;

قائمة فحص سريعة (Checklist)

  • ملفان موجودان: index.html + style.css
  • ربط CSS صحيح (<link rel="stylesheet" href="style.css">)
  • h1 + فقرة تعريف
  • صورة تظهر + alt موجود
  • قائمة ul (4 عناصر)
  • جدول مكتمل (th + td)
  • رابط داخلي يعمل (href="#form" + id موجود)
  • نموذج فيه required ويعمل submit/reset
  • CSS واضح: card + table + form
تحفيز سريع (اختياري لمن ينجز بسرعة): أضف :hover للروابط، أو caption للجدول، أو placeholder للحقول.
